# :fire: Fire Tasks

An awesome to-do list application using React, Firebase and Styled Components.

### :zap: Main features

- Add new tast
- Edit task
- Remove task
- Sort by priority

#### :heavy_check_mark: Covered concepts

- React Hooks: useState, useEffect, useContext
- Firestore

#### :package: Other dependencies

- react-toastify
- styled-icons
